# Bootstrap Icons

A shortcut to render Bootstrap Icons: https://icons.getbootstrap.com/.

## How To use it

Clock Alarm icon:

`[[` `icon:alarm` `]]` → [[icon:alarm]]

Clock Alarm with specific inline style for size and color:

`[[` `icon:alarm size="2rem" color="cornflowerblue"` `]]` → [[icon:alarm size="2rem" color="cornflowerblue"]]
